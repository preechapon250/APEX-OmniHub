# DEBUG Protocol (Universal)

1) Reproduce (steps + environment)
2) Reduce to minimal case
3) Classify: layout | state/race | rendering | data | platform | tooling
4) Instrument boundaries (event→state→render→effect)
5) Inspect (UI tree, computed values, logs)
6) Patch root cause + add regression test
7) Prevent (lint/invariant/monitor)

Output must include: root cause, patch, verification, prevention.
