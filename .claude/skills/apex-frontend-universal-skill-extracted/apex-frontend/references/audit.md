# AUDIT Protocol (Universal)

## Inputs (any 2)
analytics, support tickets, reviews, session replays, usability tests

## Friction Map
| Step | Intent | Evidence | Friction | Severity | Hypothesis | Fix | Metric |

## Prioritize (ICE)
(Impact × Confidence) / Effort

## Deliver
- Top 1–3 fixes with measurable hypotheses
- Experiment plan + guardrails + rollback
