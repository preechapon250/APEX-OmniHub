# APEX-Power Quick Reference

## Universal Execution Protocol (APEX-UEP)

```
SCOPE → CONTEXT → PLAN → EXECUTE → VERIFY
```

## Decision Router

| Task    | Protocol    |
| ------- | ----------- |
| Feature | APEX-TDD    |
| Bug     | APEX-DEBUG  |
| Design  | APEX-PLAN   |
| Review  | APEX-REVIEW |

## TDD Cycle

```
RED → VERIFY → GREEN → VERIFY → REFACTOR
```

## Debug Protocol

```
INVESTIGATE → ANALYZE → HYPOTHESIZE → IMPLEMENT
```

## Iron Laws

1. Evidence first
2. Test first
3. Verify always
4. One thing at a time
5. No rationalization

## Red Flags (STOP)

- "Maybe this will work"
- "I'll test later"
- "This is simple enough"
- "Just this once"
- "While I'm here..."
