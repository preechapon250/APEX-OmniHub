# SBOM Artifacts

Software bill of materials files are generated by CI in the `sbom-gate` job and
uploaded as build artifacts.

- `build-artifacts/sbom-npm.cdx.json`
- `build-artifacts/sbom-python.cdx.json`

They are intentionally not committed to avoid massive, noisy diffs in pull
requests.
